# Arduino-RC-Controller

## Example 1

RC Receiver   ----   Arduino Mega 2560
- Ground ----- Power Ground
- 5 V+  -------- Power 5V+ 
- CH1 --------- Communication 18TX1
- CH2 --------- Communication 19RX1
- CH3 --------- Communication 20SDA
- CH4---------- Communication 21SCL
